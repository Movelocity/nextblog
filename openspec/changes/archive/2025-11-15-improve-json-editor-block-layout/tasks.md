## 2. 响应式布局实现
- [x] 2.1 实现桌面布局（>1024px）

  - 2 个编辑框：各 50% 宽度，并排显示
  - 3 个编辑框：各 33% 宽度，并排显示
- [x] 2.2 实现平板布局（640-1024px）
  - 1-2 个编辑框：并排显示
  - 3 个编辑框：2+1 布局（前两个一行，第三个单独一行）
- [x] 2.3 实现移动端布局（<640px）
  - 所有编辑框垂直堆叠（单列）
- [x] 2.4 使用 CSS Grid 或 Flexbox 实现响应式布局
- [x] 2.5 添加 CSS transition 使布局切换更流畅
